# Vex V5 Robotics Team 3327A - Autonomous & Driver Control PROS Code 

This repository contains the PROS codebase for our team, 3327A, competitions robot for the 2025-2026 VEX VRC (V5) season.
This contains our autonomous routine & driver control. 

## To build and upload 
*if I am ever no there and you guys need to for what ever reason upload new code follow this*
- Make sure git is installed (If not follow this [Guide](https://github.com/git-guides/install-git))
- Make sure you have the pros CLI installed (If not follow this [Guide](https://pros.cs.purdue.edu/v5/getting-started/index.html))
- Clone the repo (Get the code) `git clone "https://github.com/3327A/3327A"`
- Go to the folder `cd 3327A`
- Build (compile) the code `pros build` or open the pros extension in VSCODE and hit "Build"
- Upload the code to the bot `pros upload` or using the VSCODE pros extension, hit the "Upload" button
